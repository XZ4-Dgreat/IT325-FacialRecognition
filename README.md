# IT325 - Face Recognition Authentication System

A biometric authentication system with face recognition for login and signup, built with React, TypeScript, and PHP.

## Features

- ✅ Traditional username/password authentication
- ✅ Face recognition using MediaPipe
- ✅ Two-factor authentication (password + face)
- ✅ Real-time face detection with circular progress indicator
- ✅ Mobile-responsive design
- ✅ PHP/MySQL backend for user management

## Tech Stack

### Frontend

- React 18 with TypeScript
- MediaPipe Face Detection
- React Router for navigation
- Custom CSS for styling

### Backend

- PHP 8.x
- MySQL database
- RESTful API endpoints
- Face descriptor comparison algorithm

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
